## Build Setup

``` bash
# 安装依赖关系
npm install

# 运行服务
npm run dev

# 打包与发布
npm run build

# 为生产环境构建并查看bundle analyzer报告
npm run build --report

# 运行单元测试
npm run unit

# 运行e2e测试
npm run e2e

# 运行所有测试
npm test
```
